# node-mongodb-movie

Node.js + MongoDB 网站后台增删改查的简单轮子

## 运行项目(以管理员执行)

```bash
  ## 1. 安装项目依赖
  npm install

  ## 2. 启动 MongoDB

  ## 3. 启动项目
  npm start
```

## 技术栈

- **模板引擎**  jade

- **数据库**  MongoDB

- **数据库建模工具** [mongoose](https://mongoosejs.com/docs/api.htm)
  [具体笔记请点击](./docs/mongoose.md)


- **服务器端**  node + express

- **密码加盐** [bcrypt](https://github.com/kelektiv/node.bcrypt.js)
  [具体笔记请点击](./docs/bcrypt.md)
